Installation:
npm install
npm run dev

Hinweis:
Die Kontaktformular-Funktion ist implementiert.
Für den Versand von E-Mails werden Environment Variables benötigt (EMAIL_USER, EMAIL_PASS).

Aus Sicherheitsgründen sind diese nicht im Projekt enthalten.
Im produktiven Einsatz (z. B. Deployment) funktioniert der Mailversand vollständig.
